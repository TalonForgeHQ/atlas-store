## Non-overlapping wedge

Abridge 725 owns EHR-native Best-in-KLAS ambient documentation across enterprise health systems. Nabla 726 owns the Paris/EU-trained, agentic, KLAS-A+-rated Copilot and Epic/athenahealth integration lane. Freed 727 owns the clinician-first workflow layer: AI scribe + specialty templates + visit prep + coding + EHR Push + Front Desk, with the publicly stated 98% medical-term recall across 30+ specialties and 26,000+ clinician distribution. The procurement question is whether the conversation-to-note-to-code-to-EHR-transfer-to-patient-communication chain remains attributable, reviewable, HIPAA-safe, and reversible as each workflow module changes.

## Five evidence joins

1. Encounter audio/transcript boundary → specialty template and model version → draft note → clinician edit/sign-off.
2. Clinical concept recall benchmark → suggested diagnosis/code → coding-assistant rationale → clinician override.
3. Visit-prep source set → generated prep brief → encounter decision → note and patient-instruction output.
4. EHR Push transfer → destination EHR record/version → failed-transfer retry → rollback/deletion receipt.
5. Front Desk call/message → AI action → human escalation → patient communication and audit receipt.

**Commercial safety:** Privacy, legal, security, support, press, incident, and abuse routes are excluded. No email, form submission, demo request, or revenue is claimed.
