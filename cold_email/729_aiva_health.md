## Distinct closer wedge

Its evidence challenge is joining each spoken or typed charting instruction to the correct patient, encounter, flowsheet, row, value, destination record, downstream action, human confirmation, correction, and rollback.

## Five evidence joins

1. Voice/text instruction → patient + encounter resolution → flowsheet + row → discrete value or narrative note.
2. Assistant/model version → clinical terminology interpretation → proposed chart entry → nurse confirmation/override.
3. Epic/Cerner destination + record version → write receipt → voice correction → corrected-entry receipt.
4. Smart-room / ServiceNow / Zebra action → device or workflow target → execution state → human escalation.
5. Retention/deletion request → transcript/telemetry scope → downstream deletion propagation → exportable HIPAA and EU AI Act Art. 14 receipt.
