## Five evidence joins

1. Clinician dictation / ambient audio → per-specialty model and prompt version → draft note → clinician edit/sign-off → EHR record version → reversal receipt.
2. Coding suggestion → per-code-version and per-coding-model version → clinician/auditor override → claim submission → revenue-cycle receipt across payer / clearinghouse / EHR write-back.
3. Clinical-reasoning Q&A → per-source / per-document / per-model version → cited answer → clinician adoption → order or referral action → patient-safety receipt.
4. EHR Push / write-back → destination EHR record/version (Epic / Oracle Health / athenahealth / MEDITECH) → failed-transfer retry → rollback/deletion receipt.
5. Voice-first ambient → speech-model version → accent/dialect adaptation → specialty-template adaptation → clinician override → downstream note/coding/clinical-reasoning receipt.
