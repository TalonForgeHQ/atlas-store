Arize AX

AI Engineering Platform

Arize AX is an AI engineering platform for improving AI agents and applications by observing real behavior, evaluating quality, and proving each change works before you ship. Arize AX gives you the tools to evaluate agent trajectories and decisions with real-time monitoring and automated alerting to surface failures the moment they happen. Sign up for free to get started, then follow the Arize AX workflow below to capture traces, evaluate the quality of outputs, and improve across prompts, models, and agent logic.

The Arize AX workflow

1

Instrument

Capture your first trace. Instrumentation adds tracking to your agents to capture traces: step-by-step records of agent activity (inputs, outputs, invoked tools, and costs). After instrumentation, your application streams traces into Arize AX to lay the foundation for measurable improvement.

Observe

Identify where your agent is failing. Once traces are flowing, use Signal to surface issues, or ask Alyx to surface failing runs in plain language. Look across multiple runs to spot patterns, and create dashboards and alerts to maintain visibility.

3

Evaluate

Create repeatable checks for quality. Review traces and categorize outputs with annotations. Good examples become a curated collection of test cases, while failures reveal mistakes that can be identified with evaluators: repeatable tests that score outputs against set criteria.

Create evaluators

Set up automated evals so every response is scored without manual review.

4

Improve

Test changes and measure their impact. Change the prompt, swap the model, or adjust how the app retrieves information, then run it against your test cases to verify improvement. Each run is an experiment, and Arize AX tracks every version so you can compare them and see which performs best.

Automate every step with AI using Alyx, the AI engineering agent built into Arize AX, the Arize AX CLI inside your coding agent, and the Agent Improvement Loop to instrument, observe, evaluate, and improve.

​
Alyx: Your AI engineering agent for every step

Alyx is the AI engineering agent built into Arize AX. Ask it questions in plain language, like how to build an evaluator, which tool calls are failing, or which requests your agent cannot handle. Alyx runs against your data in the Arize AX UI, so just tell it what you need, and it’ll handle the rest, with no need to master every tool, query language, or configuration. Learn more in the Alyx documentation.
