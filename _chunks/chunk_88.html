4. Five Audit Gaps in a Naive Chroma Audit Setup

Gap 1 — End-to-end provenance missing. A setup that captures vector-upsert + vector-read but not metadata-filter-decisions + hybrid-search-decisions + scalar-quantization-events fails column 4 onward. The fix is to instrument Chroma's $rank fusion hybrid-search path + every quantization event + every HNSW rebuild with audit hooks.

Gap 2 — OSS / Cloud / Enterprise provenance blurred. Chroma ships Apache-2.0 OSS + Chroma Cloud proprietary control-plane + Chroma Enterprise. EU AI Act Art. 53(d) GPAI training-data transparency + Art. 53(1)(b) copyright-summary + Art. 53(2) publicly-available-summary require the OSS / Cloud / Enterprise boundary to be explicit in the evidence packet. Naive setups mix the three.

Gap 3 — Prompt-injection + retrieval-source-poisoning + metadata-filter-bypass + hybrid-search-score-manipulation + multimodal-embedding-poisoning undetected. OWASP LLM Top 10 LLM01 (prompt injection) + LLM06 (sensitive information disclosure) + NIST AI RMF MEASURE + EU AI Act Art. 9 risk-management + Art. 14 human-oversight require detection across BM25 text signals, dense-vector embeddings, scalar/binary/product quantization signals, HNSW graph mutations, snapshot events, API-key-rotation events, RBAC-role-change events, SSO-SAML-assertion events, BYO-key-rotation events, CMEK-key-rotation events, VPC-peering events, PrivateLink events, metadata-filter inputs, multimodal-embedding inputs and LLM downstream call outputs. Naive setups instrument only the LLM call.

Gap 4 — Cross-tenant isolation evidence missing across deployment modes. SOC 2 CC6.1 + GDPR Art. 28 + HIPAA + EU AI Act Art. 10 require isolation-evidence per collection ACL + per-collection auth token + per-tenant API key + RBAC role change + snapshot export + HNSW rebuild event + VPC peering topology + PrivateLink topology across Chroma Cloud SaaS + Chroma Enterprise + Chroma Embedded (self-hosted) + Chroma Server-mode. Naive setups capture only one deployment mode.

Gap 5 — WORM retention + cost-attribution join-table missing. SEC 17a-4 WORM + IRS 6001 + EU AI Act Annex III 4 high-risk-classification per Art. 6/14/27/43/72 + Aug 2026 GPAI enforcement require WORM retention + cost-attribution across vector writes + vector mutations + vector deletes + collection CRUD events + hybrid-search decisions + scalar/binary/product-quantization events + HNSW rebuild events + snapshot exports + API-key-rotation events + RBAC-role-change events + SSO-SAML-assertion events + BYO/CMEK-key-rotation events + VPC-peering events + PrivateLink events + downstream embeddings-call events + downstream LLM call events + downstream warehouse state-change events + downstream object-store state-change events.
